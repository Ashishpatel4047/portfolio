Multi-Modal Fashion Assistant

  • Python
  • PyTorch
  • CLIP
  • SentenceTransformer
  • Gradio

Este proyecto es un asistente de moda que puede entender tanto imágenes como texto. Usando los modelos CLIP y SentenceTransformer, el sistema puede encontrar prendas de vestir similares a una imagen o recomendarlas basándose en una descripción textual. El reto fue optimizar los embeddings para una búsqueda rápida y precisa en un gran catálogo de moda.

Text-to-Speech System (Indian Languages)

  • Python
  • TensorFlow
  • Tacotron 2
  • WaveGlow
  • Data Augmentation

Como parte de mi internado en IIT Mandi, desarrollé un sistema de Texto a Voz (TTS) de alta calidad enfocado en lenguajes de la India. Implementé y entrené modelos de deep learning como Tacotron 2 para la conversión de texto a espectrogramas y WaveGlow para la síntesis de audio. El mayor desafío fue la recopilación y preprocesamiento de datos para lenguajes con recursos limitados, logrando una síntesis de voz natural y clara.
